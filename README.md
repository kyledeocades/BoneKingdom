Bone Kingdom is a small 2D lane-based strategy game built in Godot.  
The player controls a skeleton army fighting against a human kingdom.  
The goal is to gather bones, spawn units, and destroy the enemy base.

---

## Project Overview

Current prototype features:
- Player and enemy bases
- Resource system using bones
- Worker unit for gathering bones
- Combat unit spawning
- Basic enemy auto-spawning
- Unit health and base health
- Win / lose conditions

---

## Built With

- Godot 4.x
- GDScript

---

## How to Open the Project

1. Install **Godot 4.x**
2. Clone or download this repository
3. Open Godot
4. Click **Import**
5. Navigate to the project folder
6. Select the `project.godot` file
7. Click **Import & Edit**

---

## How to Test Run the Game

1. Open the project in Godot
2. Make sure the main scene is set correctly
3. Press the **Play** button in the top right

If the main scene is not set:
- Go to **Project > Project Settings > Application > Run**
- Set the main scene to:
res://scenes/Main.tscn
